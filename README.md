# Matrix Operations in C

A C program that creates a 3x2 integer matrix and performs basic statistical operations on it.

## Features

- User input for matrix elements
- Formatted matrix display
- Calculates maximum, minimum, and sum using pointers

## How to Run

1. Compile the program:
```bash
   gcc main.c -o matrix
```
2. Run the executable:
```bash
   ./matrix
```

## Example Output
```
Enter the elements of the matrix:
1 2 3 4 5 6

The matrix is:
   1    2
   3    4
   5    6

Maximum value: 6 | Minimum value: 1 | Sum of all elements: 21
```
